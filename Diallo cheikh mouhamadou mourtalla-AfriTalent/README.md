Description

AfriTalent est une plateforme web destinée à mettre en relation les freelances africains et les clients à la recherche de compétences professionnelles. Le site permet de découvrir des profils de freelances, consulter les offres tarifaires proposées et contacter facilement l'équipe via un formulaire dédié. Le projet a été conçu avec une interface moderne, responsive et accessible sur différents appareils.

Fonctionnalités principales

- Navigation responsive

- Plans tarifaires

- FAQ interactive
- Design responsive adapté aux mobiles et ordinateurs
